## 🚀 Technologies Used
- **Kotlin** 🟡
- **Jetpack Compose** 🖌️
- **MVVM Architecture** 🏗️
- **Retrofit** 🌐
- **Hilt (Dependency Injection)** 🔧
- **Coroutines & Flow** ⚡
- **TMDb API** 🎞️

## 🔗 API Reference
This app fetches movie data from [The Movie Database (TMDb)](https://www.themoviedb.org/).

## 📌 Features
✅ Browse trending movies  
✅ View detailed movie info  
✅ Responsive UI with Jetpack Compose
